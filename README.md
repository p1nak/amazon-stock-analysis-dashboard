# 📊 Amazon Stock Analysis Dashboard

## 📌 Project Overview

This project performs **time-series analysis of Amazon stock data** using Python.
The analysis explores stock price trends, volatility, trading volume behavior, and long-term growth patterns.
An **interactive dashboard** is also created to visualize the results.

The goal of this project is to demonstrate **data analysis, visualization, and dashboard development skills** using real financial data.

---

## 🚀 Features

* 📈 **Trend Analysis** of Amazon stock closing prices
* 📉 **Daily Return & Volatility Analysis**
* 📊 **Moving Average Indicators** (30, 50, 200 days)
* 📦 **Trading Volume Analysis**
* 🔗 **Correlation Analysis** between stock variables
* 📅 **Yearly Growth Analysis**
* 🖥 **Interactive Dashboard** built with Plotly & Dash

---

## ⚙ Installation

Clone the repository:

```
git clone https://github.com/p1nak/amazon-stock-analysis-dashboard.git
```

Navigate into the project folder:

```
cd amazon-stock-analysis-dashboard
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶ Running the Dashboard

Run the dashboard script:

```
python dashboard.py
```

Then open your browser and go to:

```
http://127.0.0.1:8050
```

## 📈 Key Insights

* Amazon stock shows a **strong long-term upward trend**.
* Periods of **high volatility** correspond to market uncertainty.
* **Trading volume spikes** occur during major price movements.
* Moving averages confirm **long-term bullish momentum**.
